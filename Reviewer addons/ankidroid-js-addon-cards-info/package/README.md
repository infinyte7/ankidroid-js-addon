Show pop up in with cards info during review in AnkiDroid

## Install
1. Download unofficial AnkiDroid.JS version from [release page](https://github.com/infinyte7/Anki-Android/releases)

2. Copy ```npm i ankidroid-js-addon-cards-info```

5. Paste in AnkiDroid
    
    `
    AnkiDroid -> Addons -> Import Addons
    `

6. Wait for installation to complete

7. Change note type from options menu to view reviewer / note editor addons

8. Turn on to use in reviewer / note editor

## Usage
1. Open a deck to review
2. Click on info icon to view cards info
